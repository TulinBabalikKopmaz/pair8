

Docker model eğitirken terminale aç cmd olduğuna emin ol ve aşağıdakini yaz 

docker-compose run --rm pair8 python app/randomforest.py

sonrasında api ile bağlantıyı kurmak adına aşağıdakileri terminale yaz
Dockerfiledan hangi dosyanıyı calıştırıcaksan buraya yazmayı unutma

# During debugging, this entry point will be overridden. For more information, please refer to https://aka.ms/vscode-docker-python-debug
CMD ["uvicorn", "app.mainrf:app", "--host", "0.0.0.0", "--port", "8000", "--reload"]

sonrasında
docker build -t pair8 .

sonrada
docker run --env-file .env -p 8000:8000 pair8

eğer aşağıdaki geliyorsa iyi

Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
